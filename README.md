:ru: **ru**


:pushpin: Структура папок:
```
Все папки - отдельные небольшие проекты.
Наименование папки проекта: p###-####
Где ### - номер проекта, начиная с 001
и #### - наименование проекта, разделенное символами тире (-)
```


:pushpin: Коммиты в git:
```
Коммиты в отдельные проекты начинаются с префикса p###
```


:pushpin: Структура отдельного проекта:
```
Папки /build, /bin, /target используются для компиляции и сборки проекта (потому не должны попасть в гит)
Файл INFO.md - содержит инфирмацию и описание проекта
Файл NOTES.md - содержит ваные заметки, замечания по реализации или ссылки на используемые туториалы
Файлы build.sh / clear.sh / run.sh - сборка / очистка / запуск проекта
```

:gb: **en**


:pushpin: Folder structure:
```
All directories are separate small projects.
Project directory naming: p###-####
Where ### - project number, starts with 001
and #### - project name, separated by dashes (-)
```


:pushpin: Git commits:
```
Project commit names starts with prefix p###
```


:pushpin: Project folder structure:
```
The /build, /bin, /target directories are used to compile and build the project (therefore they should not be included in git)
The INFO.md file contains project's information and description.
The NOTES.md file contains important notes, implementation comments or links to tutorials used
The build.sh / clear.sh / run.sh files are used to build / to clear / to run the project.
```
